# BrightLearn-DATE-FUNCTIONS-Exercise

## Overview

This exercise was completed during a classroom session where we followed our instructor step by step while learning how to work with Date Functions in Databricks. The lesson focused on understanding how dates can be manipulated, formatted, and analyzed using SQL queries.

The practical approach allowed us to write queries alongside the instructor and immediately see the results, making it easier to understand how Date Functions are used in real world data analysis scenarios.

## Skills Covered

During this exercise, we learned how to:

* Work with different Date Functions in SQL
* Extract specific parts of dates such as year, month, and day
* Perform date calculations and transformations
* Analyze date related data more effectively
* Create visualizations from query results
* Build dashboards to present insights in a clear and meaningful way

## Tools Used

* Databricks
* SQL Editor on databricks
* Databricks Visualizations
* Databricks Dashboards
* GitHub

## What I Learned

This was an enjoyable and practical lesson that helped me strengthen my understanding of working with dates in SQL. I gained valuable experience in querying data, transforming date values, and presenting results through visualizations and dashboards.

One of the most useful parts of the exercise was learning how data can be transformed into meaningful visual presentations. This helped me understand not only how to analyze data but also how to communicate insights effectively to an audience.

## Conclusion

This exercise provided hands on experience with Date Functions in Databricks and reinforced the importance of data visualization and dashboard creation. The skills gained from this lesson will be useful for future data analysis projects and reporting tasks.
